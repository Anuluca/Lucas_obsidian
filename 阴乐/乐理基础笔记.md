## 一、 音名

**音名**是音乐中最基础的命名系统，用来表示音高。

**音高（Pitch）** 指的是声音的实际频率（赫兹 Hz），决定了音的“高低”，比如**A4 = 440Hz**（标准音高）。

* * *

### 1. 基本概念

- **定义**：音名是用来表示音高的字母名称，通常用英文字母表示。
- **基本音名**：C、D、E、F、G、A、B。
  - 这是国际通用的音名系统，也是常说的 **“自然大调音阶”**（白键）。

* * *

### 2. 音高

- **音名对应的音高**：

  - 音名不仅仅有C、D、E、F、G、A、B这七个基本音，它们还可以通过 **“升高”** 或 **“降低”** 来产生更多的音。这些音用 **“升号（#）”** 和 **“降号（b）”** 来表示。

- **音高顺序**：

  - 从低到高：C、C♯/D♭、D、D♯/E♭、E、F、F♯/G♭、G、G♯/A♭、A、A♯/B♭、B、C（下一个八度）。
  - ![[1.png]]

- **升降号的念法**：

  - **C#**念作**升C**，**Cb**念作**降C**。

- **等音**
  - **C#**和**Cb**代表同一个音高，称为等音。
  - **B#**和**C**也是等音
- **重升和重降**：

  - **重升（𝄪）**指升两次半音
  - **重降（𝄫）**指降两次半音

- **还原号**：

  - **还原号（Natural Sign，符号：♮）** 取消音符的**升、降、重升（𝄪）、重降（𝄫）**，使其回到自然音高。

  - **作用1. 取消调号中的升降音**

    - **调号**：乐谱开头标注的升降号会影响整首曲子的对应音符。

      - 例如，G大调的调号是 **F♯**（所有F音默认升半音）。

    - **还原号的作用**：如果在某个小节中需要临时取消调号的升/降，可以用还原号。

      - 例子：在G大调中，某小节的F音需要弹成自然F（而非F♯），则会在F前加 **♮**。

  - **作用2. 取消临时的升/降号**

    - **临时升降号**：在同一小节内，若某个音被临时升/降（如 **♯**、**♭**、**𝄪**、**𝄫**），之后若需要恢复本位音，则用还原号。

      - 例子：

        - 第1个音：**F♯**（升F）

        - 第2个音在第二个小节：**F♮**（取消升号，回到自然F）

  - **作用范围**

    - **仅限当前小节**：还原号的效果只持续到当前小节结束，之后音符会恢复调号或默认音高。

      - 例子：

        - 第1小节：**F♯ → F♮**

        - 第2小节：F音恢复为调号规定的F♯（如果是G大调）。

* * *

### 3. 音名循环性

从一个C键到下一个C键，就是 **一个八度** 的距离。

* * *

### 4. 音名在音乐中的应用

- **谱子中的音名**：在乐谱中，音名通过五线谱来表示。不同的线和间表示不同的音名，通过谱号（如高音谱号、低音谱号）来确定音名的具体位置。

- **调式中的音名**：不同的调式（如C大调、G大调、F大调等）会使用不同的音名组合。比如，C大调的音阶是C、D、E、F、G、A、B，而G大调的音阶是G、A、B、C、D、E、F#。

### 5. 主音、上主音、中音、下属音、属音、下中音、导音

在音乐理论中，**主音、上主音、下属音、属音、导音**等术语通常用于描述音阶中各个音符的功能，特别是在**调性**和**和声**分析中。它们有助于理解音乐的结构和和声关系。

负责**稳定音阶**、**推动和弦的变化**、**形成和声的支撑结构**等作用

拿自然大调举例：

| 主音  | 上主音 | 中音  | 下属音 | 属音  | 下中音 | 导音  |
| --- | --- | --- | --- | --- | --- | --- |
| C   | D   | E   | F   | G   | A   | B   |

**1. 主音（C）**

- **主音**是调式中的“中心音”，它是音阶的第一个音，且所有其他音符都围绕它构建。
- **主音**代表调性稳定的中心，是乐曲的“归宿点”，常常作为和弦的根音。

**2. 上主音（D）**

- **上主音**通常具有强烈的向主音（第 1 音）归属的趋势，因此叫做“导音”。
- **上主音**和主音之间的音程非常紧密（通常是半音），这使得它在和声和旋律中起到强烈的引导作用。

**3. 中音（E）**

- **中音**是音阶中的第三个音，它位于主音与属音之间。
- **中音**具有中等的稳定性，既不像主音那么安定，也不像属音那样需要向主音归属。

**4. 下属音（F）**

- **下属音**是音阶中的第四个音，它与主音的关系比较紧张，常常向主音（Tonic）发展。
- **下属音**与属音（Dominant）关系紧密，是和弦进行中的重要组成部分，常常用于建立和声的转折。

**5. 属音（G）**

- **属音**是音阶中的第五个音，是一个非常重要的和声构成音。
- **属音**与主音之间有强烈的关系，属和弦（Dominant Chord）通常会解决定归向主和弦，形成**主属关系**。

**6. 下中音（A）**

- **下中音**是音阶中的第六个音，位于主音的下方，它和中音的关系较为接近。
- **下中音**有时会作为副主音使用，也可以用来引导向主音的过渡。

**7. 导音（B）**

- **导音**的作用和功能与上面提到的上主音一致，具有强烈的归属感，向主音引导。
- 在和声上，**导音**会带来强烈的“导向”效果，使得旋律产生向主音归属的动力。

## 二、 音程

**音程**是指两个音之间的相对高低关系，用数字表示，单位为**“度”**。比如，从C到D是一个二度音程，从C到E是一个三度音程。音程的大小可以是**全音**、**半音**，也可以是更大或更小的音程。

* * *

### 1. 半音

- **定义**：半音是音与音之间最小的距离单位。在钢琴键盘上，相邻的两个键（无论是白键还是黑键）之间的距离就是一个半音。

- **举例**：

  - 从C到C#（或Db）是一个半音。
  - 从G到Gb（或F#）也是一个半音。
  - 从E到F也是一个半音。

- **特点**：半音是音与音之间最紧密的联系，听起来非常接近。

* * *

### 2. 全音

- **定义**：全音是由两个半音组成的距离。换句话说，全音是比半音大一倍的距离。

- **举例**：

  - 从C到D是一个全音。

  - 从D到E是一个全音。

  - 从E到F#是一个全音。

- **特点**：全音比半音更宽，听起来也更有距离感。

### 3. 如何数音数

1. **C**到**E**是**3度**，数法是C和D之间到**白键数量**，**包括其自身**。
2. **C#**到**E**也是**3度**，数法是**先把C#降到C**，再按基本方法数数量。
3. **C**到**Ed**也是**3度**，数法是**先把Ed降到E**，再按基本方法数数量。

### 4. 音程的分类

**音程可以按照品质分类**，包括**纯、大、小、减、倍减、增、倍增**。

音程是由**音数（度数）**和**音程性质（品质）**决定的：

- **音数（度数）**：指的是两个音在音阶上的相对位置，比如 **C→E 是三度**，**C→G 是五度**。
- **音程性质（品质）**：指的是音程在半音上的具体间距，比如 **C→E 在大调音阶中是大三度（Major 3rd）**，**C→Eb 是小三度**。

### 5. 纯（P）、大（M）、小（m）、增（A）、减（d）、倍增（AA）、倍减（dd）的区别

| **类别**       | **包含的音程**   | **特点**  |
| ------------ | ----------- | ------- |
| **纯音程（P）**   | 1度、4度、5度、8度 | 稳定，不分大小 |
| **大音程（M）**   | 2度、3度、6度、7度 | 在大调中出现  |
| **小音程（m）**   | 2度、3度、6度、7度 | 比大音程少半音 |
| **增音程（A）**   | 纯/大音程升高半音   | 更紧张     |
| **减音程（d）**   | 纯/小音程降低半音   | 更紧张     |
| **倍增音程（AA）** | 增音程再升高半音    | 极端不稳定   |
| **倍减音程（dd）** | 减音程再降低半音    | 极端不稳定   |

**1. 纯（Perfect, P）**

**定义**：音程听起来稳定、和谐，**不会有大小之分**。
**包含**：**一度、四度、五度、八度**（P1, P4, P5, P8）。

- **例子**：
  - **C → C** = 纯一度（P1）
  - **C → F** = 纯四度（P4）
  - **C → G** = 纯五度（P5）
  - **C → C（高八度）** = 纯八度（P8）

**2. 大（Major, M）**

**定义**：在大调音阶中，二、三、六、七度音程通常是“大音程”。
**包含**：**大二度、大三度、大六度、大七度**（M2, M3, M6, M7）

- **例子**：
  - **C → D** = 大二度（M2）
  - **C → E** = 大三度（M3）
  - **C → A** = 大六度（M6）
  - **C → B** = 大七度（M7）

**3. 小（Minor, m）**

**定义**：比大音程少 **一个半音**，听起来更悲伤或忧郁。
**包含**：**小二度、小三度、小六度、小七度**（m2, m3, m6, m7）

- **例子**：
  - **C → D♭** = 小二度（m2）
  - **C → E♭** = 小三度（m3）
  - **C → A♭** = 小六度（m6）
  - **C → B♭** = 小七度（m7）

**4. 增音程（Augmented, A）**

**定义**：在**纯音程或大音程的基础上升高一个半音**。

- **例子**：
  - **C → F♯** = 增四度（A4，也称“三全音”或“恶魔音”）
  - **C → G♯** = 增五度（A5）
  - **C → E#（等于 F）** = 增三度（A3）
  - **C → B#（等于 C）** = 增七度（A7）

**5. 减音程（Diminished, d）**

**定义**：在**纯音程或小音程的基础上降低一个半音**。

- **例子**：
  - **C → G♭** = 减五度（d5）
  - **C → F♭（等于 E）** = 减四度（d4）
  - **C → B♭♭（等于 A）** = 减七度（d7）

**6. 倍增音程（Doubly Augmented, AA）**

**定义**：在增音程的基础上再**升高一个半音**。

- **例子**：
  - **C → F𝄪（等于 G）** = 倍增四度（AA4）
  - **C → G𝄪（等于 A）** = 倍增五度（AA5）

**7. 倍减音程（Doubly Diminished, dd）**

**定义**：在减音程的基础上再**降低一个半音**。

- **例子**：
  - **C → G♭♭（等于 F）** = 倍减五度（dd5）
  - **C → F♭♭（等于 D）** = 倍减四度（dd4）

### 6.对音程进行分类的作用

主要用于**帮助理解和声**，和弦的构成依赖音程，比如 **大三度 + 小三度 = 大三和弦**，纯五度（P5）是三和弦的基础音程，增四度（A4）和减五度（d5）用于不稳定和弦。

## 三、 时值

**时值**是指**一个音符的持续时长**，它决定了音的长短。乐理中常用**不同的音符符号**来表示时值的长短，并且它们有严格的比例关系。

* * *

### 1. 基本时值

在常见的 **4/4 拍** 中（每小节有 4 拍，每拍是四分音符），不同音符的时值如下：

| 音符        | 时值    | 占用时长         |
| --------- | ----- | ------------ |
| **全音符**   | 4 拍   | 一整个小节（4/4拍中） |
| **二分音符**  | 2 拍   | 半个小节         |
| **四分音符**  | 1 拍   | 1/4 小节       |
| **八分音符**  | 1/2 拍 | 1/8 小节       |
| **十六分音符** | 1/4 拍 | 1/16 小节      |

**比例关系：**

- **全音符 = 2 个二分音符 = 4 个四分音符 = 8 个八分音符 = 16 个十六分音符**，音符时值按 2:1 关系缩短（全 → 二分 → 四分 → 八分 → 十六分）。
- **时值越短，音符形状越复杂**（更多的符尾或连线）。

### 2. 休止符

休止符表示**静音**，它们的时值与音符相同。

### 3. 附点音符

附点（.）表示**延长原时值的一半**。

| 音符         | 时值计算                | 例子  |
| ---------- | ------------------- | --- |
| **附点二分音符** | 2 + 1 = 3 拍         | 𝅗𝅥. |
| **附点四分音符** | 1 + 0.5 = 1.5 拍     | 𝅘𝅥. |
| **附点八分音符** | 0.5 + 0.25 = 0.75 拍 | 𝅘𝅥𝅮. |

**例子**（4/4拍）：

- **附点二分音符（𝅗𝅥.）= 3 拍**（相当于二分音符 + 四分音符）。
- **附点四分音符（𝅘𝅥.）= 1.5 拍**（相当于四分音符 + 八分音符）。

### 4. 连音符

**连音符（Tie）** 是一个弧线，连接两个相同的音符，使它们的时值相加。

![](3.png)

✅ **例子**：

- **四分音符 + 四分音符（用连音符连接）= 2 拍**
- **二分音符 + 八分音符（连音符连接）= 2.5 拍**

#### a. 连音的作用

连音的作用是**在原本均等划分的节奏中塞进不同数量的音符**。

#### b. 常见的连音种类

| **连音名称**     | **节奏特点**         | **举例**               |
| ------------ | ---------------- | -------------------- |
| **二连音**      | 在三拍子的基础上塞进 2 个音  | 6/8 拍子中用 2 个音代替 3 个音 |
| **三连音（最常见）** | 在 2 份的时间里塞进 3 个音 | 爵士、R&B、古典乐常见         |
| **四连音**      | 在 3 份的时间里塞进 4 个音 | 多见于复杂节奏              |
| **五连音**      | 在 4 份时间里塞进 5 个音  | 现代音乐、实验音乐            |
| **六连音**      | 在 4 份时间里塞进 6 个音  | 古典乐快板、鼓点编排           |
| **七连音**      | 在 4 份时间里塞进 7 个音  | 现代音乐、爵士鼓点            |
| **九连音**      | 在 4 份时间里塞进 9 个音  | 复杂节奏                 |

![三连音符号](2.png)

#### c. 练习方法

**1. 使用节拍器**

- 设定 **60 BPM**，先练习正常节奏，然后加入连音。

**2. 口述数拍**

- **三连音**：**Ta-Ka-Ta**
- **四连音**：**Ta-Ka-Ta-Ka**
- **五连音**：**Ta-Ka-Ta-Ka-Ta**
- **六连音**：**Ta-Ka-Ta-Ka-Ta-Ka**

**3. 跟音乐练习**

- **三连音**：《We Will Rock You》 - Queen
- **六连音**：古典音乐、吉他 Solo
- **五连音 & 七连音**：实验音乐、爵士鼓点

## 四、 拍号

**拍号**是乐谱开头的**两个数字**，用来表示**音乐的节奏结构**。它决定了：

1. **每小节有几拍**（小节的长度）。
2. **每一拍的时值**（用哪种音符代表一拍）。

* * *

### 1. 拍号的结构

拍号通常写在**谱号的右侧**，由**上下两个数字**组成，**上面的数字决定 “几拍”，下面的数字决定 “用什么音符当一拍”**。

**常见拍号举例**

| 拍号      | 解释                      |
| ------- | ----------------------- |
| **4/4** | 每小节 4 拍，每拍是四分音符（最常见）    |
| **3/4** | 每小节 3 拍，每拍是四分音符（圆舞曲常见）  |
| **2/4** | 每小节 2 拍，每拍是四分音符         |
| **6/8** | 每小节 6 拍，每拍是八分音符（有强弱节奏感） |

### 2. 常见拍号

- **1. 单拍子：每一拍可以直接分成两等份（二分结构）**
- 常见的 **单拍子** 有：

  - **2/4 拍**（两拍子）
  - **3/4 拍**（三拍子）
  - **4/4 拍**（四拍子）【最常见】

- **例子：**

  - **4/4 拍（四四拍）**

    - 每小节 4 拍，每拍是四分音符。
    - 强弱关系：**强-弱-次强-弱**（1 强 2 弱 3 次强 4 弱）。

  - **3/4 拍（三四拍）**
    - 每小节 3 拍，每拍是四分音符。
    - 强弱关系：**强-弱-弱**（1 强 2 弱 3 弱）。
    - 适合华尔兹舞曲，比如《蓝色多瑙河》。

- **2. 复拍子：每一拍可以再细分成 3 个小节奏（即三连音结构），有摇摆感。**
- 常见的 **复拍子** 有：

  - **6/8 拍**（最常见的复拍子）
  - 9/8 拍
  - 12/8 拍

- **例子：**

  - **6/8 拍（六八拍）**

    - 每小节 6 拍，每拍是八分音符，但音乐上通常听成 **两组三连音**（即 **强-弱-弱 | 强-弱-弱**）。
    - 适用于**轻快的进行曲**，比如《威廉·退尔序曲》。

  - **9/8 拍（九八拍）**
    - 每小节 9 拍，每拍是八分音符，通常听成 **三组三连音**。
    - 适用于 **慢板、抒情音乐**，比如肖邦的某些夜曲。

- **3. 复杂拍子 & 混合拍子：不规则拍号**
  - 例如：
    - **5/4 拍**（如电影《碟中谍》主题曲）
    - **7/8 拍**（如某些巴尔干民间音乐）
    - **混合拍号**（如 6/8 + 3/4 交替使用）

### 3. 如何数拍子

- **单拍子（4/4、3/4、2/4）**

  - 直接数：**1、2、3、4**（4/4）
  - 或者：**1、2、3**（3/4）

- **复拍子（6/8、9/8、12/8）**

  - 以每 3 个八分音符为一组，数：
          **1（强） 2（弱） 3（弱） | 4（强） 5（弱） 6（弱）**

- **奇数拍子（5/4、7/8）**
  - 可以拆成熟悉的组合，例如： **5/4 = 3/4 + 2/4**（1-2-3 | 4-5）

### 4. 如何选择拍号？

- 不同的音乐风格适合不同的拍号：
  - **4/4 拍** ：流行、摇滚、R&B（最常见）
  - **3/4 拍** ：华尔兹、民谣
  - **6/8 拍** ：进行曲、轻快的歌曲
  - **5/4 拍** ：电影配乐（如《碟中谍》）
  - **7/8 拍** ：民族音乐、前卫摇滚

## 五、 BPM（每分钟节拍数）

**BPM** （每分钟节拍数）是用来衡量音乐速度（**节奏快慢**）的单位，表示**每分钟有多少个基本拍子**，BPM数值大则音乐快，反之则慢。

* * *

### 1. BPM 的基本概念

**简单换算**：

- **BPM = 60**，表示 **每秒 1 拍**。
- **BPM = 120**，表示 **每秒 2 拍**。
- **BPM = 180**，表示 **每秒 3 拍**（非常快）。

**示例**

| BPM  | 速度描述             | 例子          |
| ---- | ---------------- | ----------- |
| 40   | 很慢（Largo）        | 某些古典慢板      |
| 60   | 稍慢（Adagio）       | 抒情钢琴曲       |
| 80   | 中等（Andante）      | 轻音乐、慢节奏流行歌  |
| 100  | 适中（Moderato）     | 大部分流行歌曲     |
| 120  | 快速（Allegro）      | 摇滚、电子舞曲     |
| 160  | 很快（Presto）       | 电子音乐、金属乐    |
| 200+ | 超级快（Prestissimo） | 极限金属、某些电子舞曲 |

### 2. BPM 在乐谱中的标记

BPM **通常写在乐谱的开头**，一般有以下几种表示方法：

- **1️. 直接写 BPM 数值**

  - 例如：**♩= 120** （四分音符 = 每分钟 120 拍）

- **2. 使用意大利术语**（传统古典乐用法）

  - **Largo（慢板）：40-60 BPM**
  - **Adagio（柔板）：66-76 BPM**
  - **Andante（行板）：76-108 BPM**
  - **Moderato（中板）：108-120 BPM**
  - **Allegro（快板）：120-168 BPM**
  - **Presto（急板）：168-200 BPM**
  - **Prestissimo（极快板）：200+ BPM**

现代流行音乐一般直接用 BPM 数值，而古典音乐会用意大利术语

### 3. BPM 代表的音乐风格

不同的 BPM 适用于不同的音乐风格，比如：

- **流行歌曲**：一般在 **80-120 BPM**（适中，容易哼唱）
- **嘻哈 / 说唱**：多在 **70-100 BPM**（节奏感强）
- **电子舞曲（EDM）**：多在 **120-140 BPM**（节奏感强，适合跳舞）
- **摇滚 / 金属**：多在 **130-180 BPM**（更有冲击力）
- **鼓打贝斯（Drum & Bass）**：多在 **160-180 BPM**（非常快）
- **古典音乐**：速度范围更广，从 **40 BPM（慢板）到 200+ BPM（快板）**

  **示例**

| BPM     | 适用音乐风格         |
| ------- | -------------- |
| 60-80   | 抒情、慢歌、爵士、古典慢板  |
| 90-110  | 说唱、流行、R&B      |
| 120-140 | 摇滚、电子舞曲、流行快歌   |
| 160-180 | 鼓打贝斯、金属、快节奏电子乐 |

### 4. 如何测量 BPM？

可以**手动数拍**或者使用**节拍器/DAW 软件**。

手动数拍的方法：

- **找准节拍**，听 15 秒内的拍子数，乘以 4（即 **拍子数 × 4**）。
- 例如：15 秒内数到 30 拍，BPM = **30 × 4 = 120**。

### 5. BPM 在实际音乐演奏中的应用

1. **调节节奏感**

- 练习乐器时，使用**节拍器**设定 BPM，保持演奏稳定。
- 例如：初学者练钢琴，可以从 **BPM = 60** 练起，再逐步加速。

2. **改变音乐风格**

- **同一首歌，不同 BPM 会产生完全不同的感觉**
- 例如：《卡农》在 **BPM 60** 是一首抒情慢曲，在 **BPM 120** 变成欢快的改编曲。

3. **混音（DJ / 电子音乐）**

- DJ 需要匹配两首歌的 BPM，确保节奏同步。
- 例如：一首 128 BPM 的 House 音乐，和一首 125 BPM 的 Techno，需要调整速度才能无缝混音。

## 六、 调式

**调式（Mode）** 是**一组音按照特定的排列方式形成的音阶**，不同的调式决定了一首音乐的风格、情绪和氛围。

**每个调式都有自己的音阶规则**，即**全音（W）与半音（H）的排列顺序**

* * *

### 1. 主要的调式分类

**常见的 7 种调式**：

| 调式                 | 别名        | 音阶结构（英文）                      | 音阶结构（中文）                      | 音阶第几级 |
| ------------------ | --------- | ----------------------------- | ----------------------------- | ----- |
| **1. 伊奥尼亚调式（最常见）** | **自然大调**  | **W - W - H - W - W - W - H** | **全 - 全 - 半 - 全 - 全 - 全 - 半** | **1** |
| **2. 多利安调式**       | **爵士小调**  | W - H - W - W - W - H - W     | 全 - 半 - 全 - 全 - 全 - 半 - 全     | **2** |
| **3. 弗里吉亚调式**      | **西班牙调式** | H - W - W - W - H - W - W     | 半 - 全 - 全 - 全 - 半 - 全 - 全     | **3** |
| **4. 利底亚调式**       | **梦幻大调**  | W - W - W - H - W - W - H     | 全 - 全 - 全 - 半 - 全 - 全 - 半     | **4** |
| **5. 米克索利底亚调式**    | **布鲁斯大调** | W - W - H - W - W - H - W     | 全 - 全 - 半 - 全 - 全 - 半 - 全     | **5** |
| **6. 艾奥利亚调式**      | **自然小调**  | **W - H - W - W - H - W - W** | **全 - 半 - 全 - 全 - 半 - 全 - 全** | **6** |
| **7. 洛克里亚调式**      | **极端不稳定** | H - W - W - H - W - W - W     | 半 - 全 - 全 - 半 - 全 - 全 - 全     | **7** |

### 2. 详细解析 7 种常见调式

**1. 伊奥尼亚调式——大调**

- **音阶：C - D - E - F - G - A - B - C**
- **特点**：明亮、欢快、稳定。
- **应用**：流行音乐、古典音乐、儿童歌曲。
- **代表曲目**：《Twinkle Twinkle Little Star》

**2. 多利安调式——爵士小调**

- **音阶：C - D - E♭ - F - G - A - B♭ - C**
- **特点**：比大调忧郁，但比自然小调更温暖。
- **应用**：爵士乐、布鲁斯、放克。
- **代表曲目**：《So What》- Miles Davis（爵士）

**3. 弗里吉亚调式——西班牙调式**

- **音阶：C - D♭ - E - F - G - A♭ - B♭ - C**
- **特点**：带有异域风情，富有戏剧性。
- **应用**：弗拉明戈、金属、电子乐。
- **代表曲目**：《Wherever I May Roam》- Metallica

**4. 利底亚调式——梦幻大调**

- **音阶：C - D - E - F# - G - A - B - C**
- **特点**：比大调更“梦幻”，带有漂浮感。
- **应用**：配乐、实验音乐、电影音乐。
- **代表曲目**：《E.T.》- Katy Perry（副歌部分）

**5. 米克索利底亚调式——布鲁斯大调**

- **音阶：C - D - E - F - G - A - B♭ - C**
- **特点**：大调但带有“蓝色”音，适用于摇滚、布鲁斯。
- **应用**：摇滚、乡村音乐、布鲁斯。
- **代表曲目**：《Sweet Child O’ Mine》- Guns N' Roses

**6. 艾奥利亚调式——自然小调**

- **音阶：A - B - C - D - E - F - G - A**
- **特点**：典型的小调音阶，悲伤、沉稳。
- **应用**：流行音乐、古典音乐。
- **代表曲目**：《Hello》- Adele

**7. 洛克里亚调式——极端不稳定**

- **音阶：C - D♭ - E♭ - F - G♭ - A♭ - B♭ - C**
- **特点**：极端不稳定、压抑感强，不常单独使用。
- **应用**：极端金属、实验音乐。
- **代表曲目**：少见

**也存在和声小调、五声音阶这类其他特殊调式，以后学到了再单独记录。**

### 3. 离调音、本调音

**离调音**和**本调音**用于区分音符是否属于某个调（Key）内。

#### 1. 本调音

**本调音**是指**在当前调性（Key）中自然出现的音符**，即该调的**大音阶或小音阶**中的音符。

**比如 C 大调的本调音**

C 大调（C Major）的音阶是： **C - D - E - F - G - A - B - C**

 这 7 个音符**完全属于 C 大调**，它们就是 C 大调的 **本调音**。
🎹 **钢琴上，C 大调的本调音全部是白键**。

#### 2. 离调音

**离调音**是指**不属于当前调性（Key）中的音符**，即**该调的音阶中没有的音**。

**比如 C 大调的离调音**

C 大调的本调音是 **C - D - E - F - G - A - B**，所以 **所有升号（#）或降号（b）都是 C 大调的离调音**。
例如：

- C 大调中 **F#、G#、D#、Bb** 等都不是 C 大调音阶中的音，因此它们是 **离调音**。

🎹 **钢琴上，C 大调的离调音主要是黑键**。

#### **3. 离调音的作用**

离调音在音乐中有很大的作用，它们通常用于：

1. **制造紧张感**：比如爵士乐、蓝调音乐常使用“蓝音”（Blues Notes），这些音通常是离调音，能增加音乐的特色。
2. **丰富和声**：某些和弦可能会包含离调音，使其听起来更有层次感。
3. **转调或变调**：使用离调音可以**引导音乐进入另一个调性**，常用于调式转换和转调。她身份证发了

### 4. 移调

整体的调子发生了移动，分为**升调/降调**。

## 七、 调性

调性指的是**音乐的中心音**（主音）和它所使用的音阶体系。

- **中心音（Tonic）**：整首音乐围绕的核心音，如 C 大调的 C、A 小调的 A。
- **音阶体系**：大调或小调音阶，如 C 大调（C-D-E-F-G-A-B）或 A 小调（A-B-C-D-E-F-G）。

* * *

### 1. 大调与小调

调性中最常见的调式有**大调**和**小调**。大调和小调的区别在于它们音阶的结构不同：

- **大调**：大调音阶的结构是**全、全、半、全、全、全、半**。大调通常给人**明亮、开朗、稳定**的感觉。
- **小调**：小调有三种形式，最常用的是**自然小调（Natural Minor）**，它的音阶结构是**全、半、全、全、半、全、全**。小调通常给人**忧郁、沉思、深沉**的感觉。

例如：

- **C 大调**：C - D - E - F - G - A - B - C
- **A 小调**：A - B - C - D - E - F - G - A

一首音乐通常只使用**一个主要调性**，但可以转调。

### 2. 关系大调和关系小调

**关系大调（Relative Major）**

- **关系大调**是指在 **某个小调** 的基础上构建的大调。它们的音名完全相同，唯一的区别是 **主音不同**。

- 关系大调的主音是小调的 **3 级音**。

**关系小调（Relative Minor）**

- **关系小调**是指在 **某个大调** 的基础上构建的小调。它们的音名也完全相同，唯一的区别是 **主音不同**。

- 关系小调的主音是大调的 **第 6 级音**。

**例子**

- **C 大调和 A 小调**

  - **C 大调**的音阶是：C - D - E - F - G - **A - B - C**
  - **A 小调**（A 自然小调）的音阶是：**A - B - C** - D - E - F - G - A

  **C 大调和 A 小调的音名完全相同**，但是 **C 大调的主音是 C，而 A 小调的主音是 A**。
  因此，**C 大调和 A 小调是关系大调和关系小调**。

- **G 大调和 E 小调**

  - **G 大调**的音阶是：G - A - B - C - D - **E - F# - G**
  - **E 小调**（E 自然小调）的音阶是：**E - F# - G** - A - B - C - D - E

  **G 大调和 E 小调的音符也完全相同**，但是 **G 大调的主音是 G，而E 小调的主音是 E**。
  因此，**G 大调和 E 小调是关系大调和关系小调**。

### 3. 转调

调性决定了乐曲的整体氛围。调性通常保持在一首乐曲的整个过程中，但在一些音乐作品中，调性可能发生变化，形成**转调**（Modulation）：

- **转调（Modulation）**：指的是在乐曲中由一个调性变化到另一个调性。例如，从 C 大调转调到 G 大调。这通常发生在乐曲的不同部分中，可以用来增加对比、紧张感或情绪变化。

### 4. 调性与和声的关系

调性与和声有着紧密的联系，和声的构建基于调性音阶中的和弦。通常，调性中的和弦可以分为以下几种主要类型：

- **主和弦**：由调性的主音及其相关音符构成，通常给人**稳定**的感觉。
- **属和弦**：属和弦是调性中最具有张力的和弦，通常需要向主和弦解决，形成**强烈的归属感**。
- **下属和弦**：下属和弦位于主和弦的下方，用于为主和弦提供和声支撑。

## 八、 和弦

**和弦（Chord）**是指**由三个或以上的不同音高的音符同时或接近同时演奏**而形成的组合。

**示例**：

- **C 大三和弦（C 大调和弦）**由 C - E - G 组成。
- **A 小三和弦（A 小调和弦）**由 A - C - E 组成。

* * *

### 1. 和弦的基本构成

和弦通常是**按照三度堆叠**的方式构建的，也就是说，每个和弦音与相邻音之间的音程关系是**三度**。常见的和弦包括：

- **三和弦（Triad）**：由三个音组成，如 C - E - G。
- **七和弦（Seventh Chord）**：由四个音组成，如 C - E - G - B。
- **扩展和弦（Extended Chords）**：包含九、十一、十三音的和弦，如 Cmaj9（C - E - G - B - D）。

### 2. 和弦的分类

和弦有多种分类方式，常见的分类包括三和弦、七和弦、增和弦、减和弦等。

**（1）三和弦**

三和弦是最基本的和弦类型，由根音、三度音和五度音构成。根据三度音和五度音的不同，三和弦可以分为以下几种：

| 和弦类型                       | 结构            | 示例（以 C 音为根音） |
| -------------------------- | ------------- | ------------ |
| **大三和弦（Major Triad）**      | **大三度 + 小三度** | C - E - G    |
| **小三和弦（Minor Triad）**      | **小三度 + 大三度** | C - E♭ - G   |
| **增三和弦（Augmented Triad）**  | **大三度 + 大三度** | C - E - G#   |
| **减三和弦（Diminished Triad）** | **小三度 + 小三度** | C - E♭ - G♭  |

- **大三和弦（X）**：根音到三度音是**大三度**（4个半音），三度音到五度音是**小三度**（3个半音）。例如：

  - **C大三和弦**：C - E - G

  - **G大三和弦**：G - B - D

- **小三和弦（Xm minor）**：根音到三度音是**小三度**（3个半音），三度音到五度音是**大三度**（4个半音）。例如：

  - **C小三和弦**：C - E♭ - G

  - **G小三和弦**：G - B♭ - D

- **增三和弦（Xaug augmented）**：根音到三度音是**大三度**（4个半音），三度音到五度音也是**大三度**（4个半音）。例如：

  - **C增三和弦**：C - E - G#

  - **G增三和弦**：G - B - D#

- **减三和弦（Xdim diminished）**：根音到三度音是**小三度**（3个半音），三度音到五度音也是**小三度**（3个半音）。例如：

  - **C减三和弦**：C - E♭ - G♭

  - **G减三和弦**：G - B♭ - D♭

每一个调式都有7个三和弦，可以分别用级数来表示，比如C大调：

| 音名    | 三和弦           | 级数  | 类型与功能      |
| ----- | ------------- | --- | ---------- |
| **C** | **C** - E - G | 1级  | 大三和弦（主和弦）  |
| **D** | **D** - F - A | 2级  | 小三和弦       |
| **E** | **E** - G - B | 3级  | 小三和弦       |
| **F** | **F** - A - C | 4级  | 大三和弦（下属和弦） |
| **G** | **G** - B - D | 5级  | 大三和弦（属和弦）  |
| **A** | **A** - C - E | 6级  | 小三和弦       |
| **B** | **B** - D - F | 7级  | 减三和弦       |

**（2）七和弦**

七和弦是在三和弦的基础上增加一个七度音，常见的七和弦有：

- **大七和弦**：由**大三和弦**加上**大七度**音构成。例如：

  - **C大七和弦**：C - E - G - B

  - **G大七和弦**：G - B - D - F#

- **小七和弦**：由**小三和弦**加上**小七度**音构成。例如：

  - **C小七和弦**：C - E♭ - G - B♭

  - **G小七和弦**：G - B♭ - D - F

- **属七和弦（也叫大小七和弦）**：由**大三和弦**加上**小七度**音构成。例如：

  - **C属七和弦**：C - E - G - B♭

  - **G属七和弦**：G - B - D - F

- **半减七和弦（减七和弦）**：由**减三和弦**加上**小七度**音构成。例如：

  - **C半减七和弦**：C - E♭ - G♭ - B♭

  - **G半减七和弦**：G - B♭ - D♭ - F

**（3）其他和弦**

除了三和弦和七和弦，还有一些特殊的和弦，如增和弦、减和弦、挂留和弦等：

- **增和弦**：由根音、增三度音和增五度音构成。例如：

  - **C增和弦**：C - E - G#

- **减和弦**：由根音、减三度音和减五度音构成。例如：

  - **C减和弦**：C - E♭ - G♭

- **挂留和弦**：通常是将三和弦的三度音替换为二度音或四度音。例如：

  - **C挂四和弦（Cadd4）**：C - E - G - A

  - **C挂二和弦（Cadd2）**：C - D - E - G

- **九和弦（Ninth Chords）**：包含五个音，如 C - E - G - B - D。
- **十一和弦（Eleventh Chords）**：包含六个音，如 C - E - G - B - D - F。\*   **十三和弦（Thirteenth Chords）**：包含七个音，如 C - E - G - B - D - F - A。

### 3. 根音

**根音（Root）**是和弦的**基础音**，它是构建和弦的起点，无论和弦如何转位或排列，其根音始终保持不变。

**1. 如何找到和弦的根音？**

和弦通常是**按照三度叠置**的方式构建的，根音是**最底层的那个音**（在原位时）。
即使和弦被转位（比如最低音改变），它的**根音仍然是决定和弦名称的音**。

**比如C 大三和弦**：

- **C（根音）** - E（大三度） - G（纯五度）
- 这个和弦的根音就是**C**，所以它叫做**C 大三和弦**（Cmaj）。

### 4. 和弦的功能与进行

在调性音乐中，和弦通常根据其功能分为三大类：

1. **主和弦**：
   - 主要是 1级和弦（如 C 大调中的 C 和弦）
   - 作用：稳定，作为音乐的“家”或“归属点”。
2. **下属和弦**：
   - 主要是 4 级和弦（如 C 大调中的 F 和弦）
   - 作用：起到转折、缓冲的作用。
3. **属和弦**：
   - 主要是 5 级和弦（如 C 大调中的 G 和弦）
   - 作用：有强烈的倾向性，推动音乐向主和弦解决。

通常的**和弦进行**：

- **最常见的和弦进行**：1 → 4 → 5 → 1（C → F → G → C）
- **流行音乐中常见的和弦进行**：1 → 5 → 6 → 4（C → G → Am → F）

### 5. 转位和弦

和弦的根音决定了和弦的基本结构，但如果把不同的音放在低音，就会形成不同的**和弦转位**：

- **原位**：根音在最低（如 C - E - G）。
- **第一转位**：三音放到最低（如 E - G - C）。
- **第二转位**：五音放到最低（如 G - C - E）。

**示例（C 大三和弦的不同转位）：**

- **C 大三和弦原位**：C - E - G
- **第一转位（E 在最低）**：E - G - C
- **第二转位（G 在最低）**：G - C - E

### 6. 挂留和弦

挂留和弦是一种通过**替换传统三和弦中的三度音**而形成的和弦。它通常用“Sus”表示。

通常用 **2nd（sus2）** 或 **4th（sus4）** 替换三音。它的特点是**不确定性和悬浮感**，常用于制造悬念或期待解决到普通和弦（如大三和弦或小三和弦）。

#### 1. 挂留和弦的构成

挂留和弦的构成基于三和弦，但通过替换三度音来改变和弦的性质。

**（1）挂四和弦（Sus4）**

- **构成**：由根音、四度音和五度音组成。

- **举例**：

  - **C挂四和弦（Csus4）**：C（根音）- F（四度音）- G（五度音）

  - **G挂四和弦（Gsus4）**：G（根音）- C（四度音）- D（五度音）

**（2）挂二和弦（Sus2）**

- **构成**：由根音、二度音和五度音组成。

- **举例**：

  - **C挂二和弦（Csus2）**：C（根音）- D（二度音）- G（五度音）

  - **G挂二和弦（Gsus2）**：G（根音）- A（二度音）- D（五度音）

#### 2. 挂留和弦的解析与解决

挂留和弦通常不会独立存在，而是**作为过渡和弦**，最终**解决**回到普通三和弦。

**Csus4 → Cmaj 解析**

1. **Csus4（C - F - G）**
2. **Cmaj（C - E - G）**

- **F 下降到 E**，形成 Cmaj 和弦。
- 这种**紧张 → 解决**的效果，让音乐更有方向感和期待感。

**Csus2 → Cmaj 解析**

1. **Csus2（C - D - G）**
2. **Cmaj（C - E - G）**

- **D 上升到 E**，形成 Cmaj 和弦。
- 听感比 sus4 更轻盈、灵动。

**结论**

| **和弦类型**         | **音符结构**   | **听感** | **解决方式**        |
| ---------------- | ---------- | ------ | --------------- |
| **普通大三和弦（Cmaj）** | C - E - G  | 明亮、稳定  | /               |
| **普通小三和弦（Cm）**   | C - Eb - G | 深沉、忧郁  | /               |
| **Csus2（挂二和弦）**  | C - D - G  | 轻盈、开放  | 解决到 Cmaj（D → E） |
| **Csus4（挂四和弦）**  | C - F - G  | 紧张、悬浮感 | 解决到 Cmaj（F → E） |

## 九、 其他名词

### 1. 音列

**音列**是指按照一定规则排列的一系列音高。它是音乐的基础，用于构建旋律、和声以及即兴演奏。
音列通常由 **七个或更多音级** 组成，并具有**特定的音程关系**。

### 2. 乐音/噪音/乐音体系

**乐音：**

乐音是**音高稳定、振动规律、适合音乐表现的声音**。
比如钢琴、吉他、长笛等乐器

**噪音：**

噪音是**振动不规则、音高不稳定、不适合用于旋律的声音**。
比如鼓声、掌声、雷声等

**乐音体系：**

乐音体系是**根据音高、音程等规律，把乐音组织成音阶、调式、和声等的体系**。


### 3. 音级/基本音级/变化音级

**音级：**

**乐音体系**中等每一个音都叫**音级**。

**基本音级：**

基本音级是指**在自然大小调音阶中，按照音阶顺序排列的七个音**，它们是乐曲的基础。
比如C大调中的基本音级就是“CDEFGAB”

**变化音级：**

变化音级是**在基本音级的基础上，通过升高或降低音高而产生的音级**。
比如C大调中的“C# D# F# G# A#”

变化音级中同样包括**“升音级”、“降音级”、“重升音级”、”重降音级“**
只要纸面上带了**升降号（#b）**或者**重升重降号（𝄪𝄫）**，那么它就是变化音级，哪怕**B#**和**C**虽然为等音，但是**B#**依然是变化音级。
**原因：**变化音级在概念上更**强调基本音级的升高或者下降**，而**B#**在写法上也更强调B的上升，所以是变化音级。

### 4. 首调/固定调

**首调**：

*   **“Do” 代表主音**，不同调里主音不同。
*   **C 大调的 Do 是 C，G 大调的 Do 是 G**，依此类推。
*   适合 **视唱练耳**，帮助理解旋律关系。

**固定调**：

*   **“Do” 永远是 C**，不管什么调。
*   **G 大调里 C 还是 Do**，D 还是 Re，固定不变。
*   适合 **学钢琴、绝对音感训练**，帮助记住音高。

### 5. 相对音感/绝对音感

*   **相对音感 → 能听出音和音之间的关系**（比如谁比谁高）。
*   **绝对音感 → 能直接听出音的名字**（比如“这个就是G”）。
